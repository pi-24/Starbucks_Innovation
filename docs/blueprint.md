# **App Name**: Brewtiful Dashboard

## Core Features:

- Personalized Recommendations: Suggest drinks based on weather, time of day, and user preferences, creating a tailored experience.
- Real-time Weather Integration: Fetch and display real-time weather data for the user's location, influencing drink recommendations. The AI tool will assess weather and decide what factors of it will incorporate into the recommendations.
- Favorite Order Reordering: Allow users to quickly reorder their last favorite drink.
- Lifestyle Picks: Offer curated drink selections suited for specific lifestyles in Dubai, such as desert drives or beach mornings.
- User-specific Data: Show personalized data to the user, such as username, amount of rewards, and favorite drink.

## Style Guidelines:

- Primary color: Starbucks green (#00704A) to reinforce the brand identity. The color evokes feelings of nature, freshness, and energy.
- Background color: Very light green (#F2FBF7) to create a clean, inviting, and refreshing atmosphere.
- Accent color: Light brown (#A07555) for interactive elements, providing a warm, familiar, and inviting experience.
- Headline font: 'Poppins' sans-serif, to be used for titles
- Body font: 'PT Sans' sans-serif, to be used for main texts
- Use consistent, modern icons related to weather, coffee, and user preferences (from Lucide React).
- Use a grid-based layout with rounded cards and clear sections for easy navigation.
- Subtle transitions and hover effects on cards and buttons for enhanced interactivity using framer-motion.
